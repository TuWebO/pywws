Reprocess¶

Introduzione¶

Il programma ricrea i dati di riepilogo orari, giornalieri e mensili che è creata dal programma Process.py. Esso deve essere eseguito ogni volta che si esegue l’aggiornamento a una versione più recente di pywws.

Il programma è molto semplice da usare:

python -m pywws.Reprocess data_directory

where data_directory is the location of your stored data.

Dettagli API¶

Rigenera i dati di riepilogo orari e giornalieri

usage: python -m pywws.Reprocess [options] data_dir
options are:
 -h | --help     display this help
 -v | --verbose  increase number of informative messages
data_dir is the root directory of the weather data
